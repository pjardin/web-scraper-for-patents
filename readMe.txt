This contains the webscapper for patents.

patentScraper.py is based on python3,

make sure pip3 is installed and updated:
	python3 -m pip install --upgrade pip

this program needs beautifulsoup:
	pip3 install beautifulsoup4

so to run, cd into /patents/code and 
> python3 patentScraper.py

the layout of the folders are as follows

patents/
	code/
		here contains patentScraper.py
	data/

		here contains the data generated by patentScraper.py

it automatically figures out were it is, so you don't need to update the file paths
